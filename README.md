# tombstone-design
A web front-end 


## 环境

建议新创建一个node环境, 避免 创建按照依赖时各种问题

### 安装 并 使用 node 

```shell
$ nvm install v13.11.0 (会自动切换)

$ nvm use v13.11.0

$ nvm ls (查看版本)
```

### 查看相关包

```shell
$ npm list --depth=0

$ npm list --depth=0 --global
```

### 查询并安装相关包

```shell
$ npm info emoji-regex

$ npm install --save emoji-regex@8.0.0

```
